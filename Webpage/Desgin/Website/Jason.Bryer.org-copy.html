Table of Contents

The primary focus of this website is to provide documentation for R packages I maintain. These include:

  • PSAboot Bootstrapping for Propensity Score Analysis.

  • TriMatch Propensity score matching for non-binary treatments.

  • multilevelPSA An R package for estimating and visualizing multilelel propensity score models.

  • DataCache An R package to manage data caching.

  • geocode An R package to geocode IP addresses using GeoIPâ€™s database and street addresses using Google or Yahoo.

  • ipeds An R package to interface with the Integraded Postsecondary Education Data System.

  • irutils An R package containing a collection of useful functions for Institutional Researchers.

  • likert An R package to analyze and visualize Likert based items.

  • makeR An R package for managing document building, versioning, and templating.

  • naep An R package to interface with the National Assessment of Educational Progress (NAEP) restricted use databases. This includes access any analyzing data using the replicate weights and multiple plausible values.

  • pisa A data-only R package for the 2009 Programme of International Student Assessment (PISA) conducted by Organisation for Economic Co-operation and Development (OECD).

  • qualtrics An R package to interface with the Qualtrics.com survey system.

  • Rd2markdown Functions to convert Rd help files to markdown format.

  • retention An R package for estimating and visualizing retention and completion rates. Designed primarily for online, distant education institutions, the method used can be generalized to traditional institutions.

  • Rgitbook Creating Gitbook Projects with R Markdown.

  • ruca An R package to get Rural-Urban Commuting Area (RUCA) Codes from zip codes.

  • sqlutils An R package to manage a library of SQL queries.

  • timeline An R package to create timeline figures.
